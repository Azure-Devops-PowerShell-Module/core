# core Module
## Description
A collection of modules for working with Azure DevOps Projects and Teams.

## core Cmdlets
### [Connect-AzDevOpsOrganization](Connect-AzDevOpsOrganization.md)
A function to authenticate to an Azure DevOps Organization

### [Get-AzDevOpsOperation](Get-AzDevOpsOperation.md)
A function to get the status of an Azure DevOps Operation

### [Get-AzDevOpsProcess](Get-AzDevOpsProcess.md)
A function to get proccesses defined in an Azure DevOps Project

### [Get-AzDevOpsProject](Get-AzDevOpsProject.md)
A function to get an Azure DevOps Project

### [Get-AzDevOpsProjectProperty](Get-AzDevOpsProjectProperty.md)
A function to return extended properties of an Azure DevOps Project

### [Get-AzDevOpsTeam](Get-AzDevOpsTeam.md)
A function to get one ore more Teams within an Azure DevOps Project

### [Get-AzDevOpsTeamMember](Get-AzDevOpsTeamMember.md)
A function to return the members of an Azure DevOps Project Team

### [New-AzDevOpsProject](New-AzDevOpsProject.md)
A function to create a new Azure DevOps Project

### [New-AzDevOpsTeam](New-AzDevOpsTeam.md)
A function to create a new Azure DevOps Project Team

### [Remove-AzDevOpsProject](Remove-AzDevOpsProject.md)
A function to remove an Azure DevOps Project

### [Remove-AzDevOpsTeam](Remove-AzDevOpsTeam.md)
A function to remove an Azure DevOps Project Team

### [Update-AzDevOpsProject](Update-AzDevOpsProject.md)
A function to update an Azure DevOps Project

### [Update-AzDevOpsTeam](Update-AzDevOpsTeam.md)
A function to update an Azure DevOps Project Team
